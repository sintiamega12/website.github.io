Web Development

Web Design

Belajar website bermula dari Web Design. Pada kursus ini, Anda akan belajar bagaimana membuat sebuah halaman website. Anda akan belajar pengetahuan dasar tentang website, dari mulai HTML dan CSS. Kemudian Anda juga akan belajar bagaimana membuat halaman website.

PHP Basic

Setelah menguasai Web Design, Anda dapat melanjutkan dengan mengikuti kursus PHP Basic. Pada kursus ini, Anda akan belajar bagaiman cara membuat sebuah website yang dinamis menggunakan bahasa pemrograman PHP. PHP merupakan bahasa pemrograman yang cukup populer dan mudah untuk dipelajari.

PHP - CodeIgniter

Setelah menguasai PHP, Anda bisa melanjutkan dengan kursus PHP - CodeIgniter. CodeIgniter merupakan sebuah framework MVC (Model - View - Controller) dari PHP. CodeIgniter merupakan framwork yang cukup populer dan mudah untuk dipelajari oleh pemula. Dengan menggunakan CodeIgniter, proses membuat sebuah website menjadi semakin mudah.
